# Architecture Overview — Extracted Reality

> Generated by `/ride` 2026-06-20. CODE IS TRUTH.

## System purpose

TurnTrace is an **evidence-grounded evaluation harness** for a single trading-card-game
agent against a simulator (`cabt` / Kaggle Pokémon TCG). Its design philosophy:
produce data before optimizing; freeze baselines before claiming uplift; seal
immutable run artifacts; tie every claim to a bounded **claim ceiling**.

## Four zones (runtime / offline separation, NFR-1)

```
                 └───────────────┬─────────────────────────────┘

        ┌────────────────────────┴──────────────────────────┐
        │ eval/   run_match · run_eval · validate ·          │
        │         mirror_validate · hygiene_check ·          │
        │         canonical_json                             │
        └───────┬───────────────────────────┬───────────────┘
                │ imports                    │ writes sealed run dir
   agents/runtime/ (stdlib only)            ▼
   random_legal · scripted_baseline   runs/<run_id>/  (local, git-ignored)
                                       manifest · match_results · traces ·
                                       hashes.txt · summary.csv · notes.md
                                            │ read-only
                                            ▼
        ┌──────────────────────────────────────────────────┐
        │ analysis/ (offline, stdlib only, no cabt)         │
        │  aggregate → {delta, dispersion, evidence,        │
        │               failure, replay, e2e} → docs/ledger │
        └──────────────────────────────────────────────────┘
```

Import direction is **mechanically enforced** by `tests/test_import_direction.py`:
runtime→stdlib; sim→cabt; eval→sim+runtime+analysis; analysis→analysis only.

## Primary data flow (one evaluation)

1. `sim/probe.py` confirms cabt is runnable and writes capability flags.
2. `eval/run_eval.py` allocates `run_id` + `manifest.json` (ID authority), checks
   the **immutability guard** (refuses a populated dir, exit 3) and the
   **deck-drift guard** (live deck hash must equal the frozen hash).
3. For each match index in the frozen seed set, `eval/run_match.py:play_match`
   drives `random_legal`/`scripted_baseline` via `SimAdapter`, emitting a
   `match-summary.json` + canonical `decision-trace.jsonl` (joined by `trace_hash`).
4. `eval/run_eval.py` writes `hashes.txt` provenance + `notes.md`, then delegates
   to `analysis/aggregate.py` for `summary.csv` and (only on `--deliverable`) one
   append-only `docs/ledger.md` row.
5. `analysis/*` produce comparisons/reports over sealed run dirs.

## Tech stack

- **Language:** Python (probed env: 3.14.0, win32).
- **Dependencies:** none — **stdlib only** in app code (NFR-7), verified.
- **Simulator:** cabt via the `cg` package (kaggle-environments 1.14.10), local/git-ignored.
- **Tests:** stdlib `unittest` + plain-assert checks (no pytest in app code).
- **Persistence:** flat files (JSON/JSONL/CSV/MD); no database.

## Governance invariants baked into code

- **FM-01:** a crash is `result="error"`, never counted as a loss (`run_match.py:162`).
- **Immutability:** sealed run dirs are never overwritten (`run_eval.py:138`).
- **Seed uncontrolled:** `seed_controlled=false` → reproducibility is
  distribution-stable + audit-trail (trace_hash recompute), not byte-identical;
  the determinism smoke is skipped.
- **Claim ceiling:** standing **Rung 2 — "beats random-legal"** (earned Cycle-007,
  `docs/claim-ceiling.md`); ledger is the only ceiling-bearing artifact; forbidden
  agent words (strong/competitive/optimal/calibrated/complete) are gated.
- **Competition-Data containment:** raw card data never enters a tracked artifact
  (adapter digests + hygiene_check + evidence_summary validator).

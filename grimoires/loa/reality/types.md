# Types / Artifact Schemas — Extracted Reality

> TurnTrace has no class-heavy type system; its "types" are the flat-file run-dir
> artifacts whose field lists are the contract. Authority: `eval/schemas.md`,
> enforced by `eval/validate.py`. Generated by `/ride`.

## Run directory layout (one sealed run)

```
runs/<run_id>/
├── manifest.json            ID authority (written FIRST)
├── match_results/<id>.json  one match-summary per match
├── traces/<id>.jsonl        one decision-trace sidecar per match
├── hashes.txt               provenance stamp (KEY=VALUE)
├── notes.md                 mode + claim ceiling note
└── summary.csv              aggregate (written by analysis/aggregate.py)
```

## match-summary.json (`eval/run_match.py:190`, schema `eval/schemas.md`)

Fields: `run_id, match_id, regime_id, experiment_id, agent_id, agent_version,
opponent_id, deck_id, opponent_deck_id, seed, seed_controlled, match_index,
result, ending_cause, turns, timeout, invalid_action_count,
invalid_action_detectable, total_decisions, trace_present, trace_hash,
started_at, completed_at, wall_clock_ms, simulator_version, sim_version_source,
deck_hash, error, notes`.

Invariants (validator): `result=="error" ⟺ error populated` (FM-01 masquerade
guard); `seed_controlled=true ⇒ integer seed`; `invalid_action_detectable=true ⇒
integer invalid_action_count`; `trace_present=true ⇒ string trace_hash + integer
total_decisions`. `result ∈ {win, loss, draw, error}`.

## decision-trace.jsonl — two record types

**Decision row** (`eval/run_match.py:121`): `record_type="decision", run_id, match_id,
regime_id, decision_index, turn, player, phase, public_state_summary,
private_state_summary, legal_actions_count, legal_actions_digest,
legal_actions_sample, selected_action{indices,option_types}, selected_action_type,
decision_latency_ms, random_seed, error, post_decision_observation`.
Invariant: opponent rows have **null** `private_state_summary` (no hidden-state leakage).

**Terminal row** (`eval/run_match.py:177`): `record_type="terminal", ..., result,
ending_cause, turns, final_prize_counts, last_decision_index`.

`trace_hash` = `hash_canonical(trace)` over the full row list — the summary↔trace
join (AC-3); recomputable from the sidecar (`recompute_trace_hash`).

## manifest.json (`eval/run_eval.py:177`)

`run_id, regime_id, expected_match_ids, match_indices, agent_id, opponent_id,
deck_a_id, deck_b_id, opponent_pool_id, seed_set_id, deck_pool_id,
metrics_spec_id, agent_version, agent_source_file, agent_source_hash,
opponent_source_hash, mode, sim_version, created_at`. `expected_match_ids` is the
immutability guard's authority.

## hashes.txt (`eval/run_eval.py:230`)

Required non-empty (AC-6): `git_rev, sim_version, seed_list_hash, timestamp`.
Also: `git_dirty`, component hashes (regime/seed_set/opponent_pool/deck_pool/
metrics_spec), `deck_hash`, and O1 source-hash provenance
(`agent_source_hash`, `opponent_source_hash`, `config_hash`) — so an agent-only,
same-regime comparison's provenance holds even with `git_dirty=true`.

## ledger.md row (`docs/ledger.md`, `analysis/aggregate.py:152`)

18 columns: `date, run_id, regime_id, git_rev, sim_version, agent_version,
opponent_pool_ref, seed_set_ref, games, win_rate, illegal_action_rate,
timeout_rate, error_rate, avg_turns, mode, hypothesis, claim_ceiling, notes`.
The **only ceiling-bearing artifact**; append-only (never edit past rows).

## Frozen-input shapes (`frozen/`)

- regime: `{regime_id, created_at, seed_set, opponent_pool, deck_pool, metrics_spec, mode, notes}` (refs by name).
- seed-set: `{seed_set_id, mode, rationale, match_indices, seeds(null=unseeded), n, n_note}`.
- opponent-pool: `{opponent_pool_id, opponents[{opponent_id, kind, agent_ref, agent_version, deck_ref, category, notes}]}`.
- deck-pool: `{deck_pool_id, note, decks[{deck_id, card_count, deck_hash, hash_algo, hash_method, source_ref}]}` (hashes only, no card data).
- metrics-spec: `{metrics_spec_id, composite_verdict, composite_note, categories{metrics[], gates{}}}` (correctness gates, no strength gate).

## Capability flags (`sim/capabilities.json`, `sim/adapter.py:41`)

`local_match_runnable` (confirmed), `legal_actions_observed` (true),
`seed_controlled` (**false**), `invalid_action_detectable` (true),
`timeout_detectable` (**false**), `own_hidden_state_observable` (true).
